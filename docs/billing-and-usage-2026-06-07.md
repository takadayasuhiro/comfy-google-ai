# API 料金・利用枠ガイド（2026/06/07）

本アプリの利用者向けに、**Google AI Pro サブスク** と **Gemini API（従量課金）** の関係、目安コスト、確認方法をまとめます。

> **重要**: Google AI Pro に加入していても、このアプリの Imagen / Veo 生成は **API キー経由の従量課金** です。Gemini アプリ（チャット等）の利用枠とは **別物** です。

---

## 1. このアプリが使う課金の仕組み

| 項目 | 内容 |
|------|------|
| 認証 | [AI Studio](https://aistudio.google.com/apikey) の **API キー**（`.env` の `GOOGLE_API_KEY`） |
| 課金方式 | **Paid Tier（従量課金）** |
| 画像 | Imagen / Gemini 画像 API |
| 動画 | Veo API（FastAPI 直結） |
| プロンプト拡張 | Gemini Flash（ごく小額） |

Google AI Pro（月額約 $20 / 日本では約 ¥2,900 前後）の契約は、**Gemini アプリの利用枠拡大** が主目的です。API 利用は別途カウントされます。

---

## 2. Google AI Pro で得られる API 関連特典

| 種類 | 内容 | 本アプリとの関係 |
|------|------|------------------|
| **Gemini アプリ枠** | チャット・Flow 等（5時間ごとリセット＋週上限） | **直接は使わない** |
| **API 向け特典** | AI Studio の上限拡大、月 **$10 GCP クレジット** | **こちらが関連** |

$10 クレジットは **自動付与ではありません**。[Google Developer Program](https://developers.google.com/program) で請求アカウントと紐付けが必要です。未連携の場合は実質使えません。

特典の確認: https://developers.google.com/program/my-benefits

---

## 3. 1回あたりの目安コスト（公式 API 単価）

出典: [Gemini API 料金](https://ai.google.dev/gemini-api/docs/pricing)（**Paid Tier**、2026/06 時点）

| 操作 | 単価の目安 |
|------|-----------|
| **Imagen 4 Standard**（デフォルト `imagen-4.0-generate-001`） | **約 $0.04 / 枚** |
| **Imagen 4 Fast** | 約 $0.02 / 枚 |
| **プロンプト拡張**（Gemini Flash） | ごく小さい（数円未満/回） |
| **Veo 3.1 Lite** 6秒 720p | **$0.05×6 ≈ $0.30 / 本** |
| **Veo 3.1 Lite** 6秒 1080p | $0.08×6 ≈ **$0.48 / 本** |
| **Veo 3.1**（高品質）6秒 | $0.40×6 ≈ **$2.40 / 本** |

- **Veo は無料枠なし**（成功した分だけ課金）
- 画像の Imagen も Paid Tier 前提
- 高品質モデル（Veo 3.1 Standard、Imagen Ultra 等）に切り替えると **数倍** のコストになります

本リポジトリのデフォルト設定はコスト最適寄りです:

- 画像: `imagen-4.0-generate-001`（Standard）
- 動画: `veo-3.1-lite-generate-preview`（Lite・最安クラス）

## 4. 月あたりのざっくり試算

### パターン A: $10 クレジットのみ（追加課金なし）

| 用途のみ | およその本数/月 |
|----------|----------------|
| 動画だけ（Lite 6秒 720p） | **約 30 本**（$10 ÷ $0.30） |
| 画像だけ（Imagen 4 Standard） | **約 250 枚**（$10 ÷ $0.04） |
| 半々くらい | 動画 15本 + 画像 120枚 程度 |

### パターン B: API に別途課金設定あり（Spend Cap など）

月 **$20〜50** くらい許容する場合の目安:

| 月予算 | 動画 Lite 6秒 | 画像 Standard |
|--------|---------------|---------------|
| $20 | 約 65 本 | 約 500 枚 |
| $50 | 約 165 本 | 約 1,250 枚 |

※プロンプト拡張・画像編集は上記よりずっと少額です。

### パターン C: 実際の「使える量」はコスト以外の制約もある

動画は **1本あたり 3〜10 分** かかるため、現実的には:

- **1日 5〜15本** 動画 → かなりヘビーな利用
- **1日 20〜50枚** 画像 → 個人開発なら十分

プレビューモデル（`*-preview`）は **RPM（分あたりリクエスト数）/ 同時実行の上限** もあるため、料金の範囲内でも連続生成は詰まることがあります。

## 5. よくある誤解

| 誤解 | 正しい理解 |
|------|-----------|
| Google AI Pro に入っていれば API は無制限 | **いいえ**。サブスク枠と API 課金は別 |
| Pro の月額 $20 が API 代に充当される | **基本的にいいえ**。API は従量。$10 クレジットは別途（要 Developer Program 連携） |
| Veo も無料枠がある | **いいえ**。Veo は成功分のみ課金 |
| 会社サーバーで動画生成すると GPU 代がかかる | Veo は **クラウド API** のためローカル GPU は不要（API 従量のみ） |

## 6. 利用前に確認すべきこと

### 管理者・導入担当者

1. **AI Studio → Billing**（https://aistudio.google.com）
   - 課金が有効か
   - **Spend Cap**（月上限）の設定値
2. **Developer Program の $10 連携**
   - https://developers.google.com/program/my-benefits
3. **利用ダッシュボード**で今月の API 使用量を確認

### 一般利用者（社員）

- 本アプリの生成は **API 従量課金** にカウントされます
- 動画は画像より **1本あたりのコスト・待ち時間** が大きいです
- 大量生成や連続リトライは控えめに（コストとレート制限の両方に影響）
- コンテンツポリシー違反のプロンプトは課金前に拒否されることもあります（[動画機能ドキュメント](video-gallery-2026-06-07.md) 参照）
